# Calorie Tracker 🍎

A responsive calorie tracking web application designed to help users monitor daily food intake and maintain healthy eating habits.

## Overview

This project was created as part of a Digital Product Management course project. The application allows users to:

- Track calories consumed
- Monitor meals throughout the day
- View nutritional information
- Navigate a clean and responsive interface

The goal of this project was to practice front-end web development, UI design, and deployment using GitHub and Azure Static Web Apps.

---

## Technologies Used

- Replit
- GitHub
- Azure Static Web Apps

---

## Features

### Daily Tracking
Users can enter meals and monitor calorie intake throughout the day.

### Responsive Design
The application is optimized for desktop and mobile viewing.

### Modern UI
Simple and clean user interface focused on usability and accessibility.
